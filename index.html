How it Works

Data is at the core of every modern business. Your data comes from multiple sources and in multiple formats, but without a unified and integrated view, you may struggle to achieve operational effectiveness or gain long-term insights. How can you make use of data that arrives in the wrong format without costly and difficult ETL functions? How can you simplify data delivery to partners and customers when each one demands a different data format?

What if there was one data integration platform that could handle multiple data inputs and outputs for you and manage all your data transformations, saving you time and effort, improving your operational efficiency, and bringing you closer to your business goals?

The Power Integrator platform – PI – is your fast-track solution for data transformation and integration, that can add the power of data to your business. PI’s flexible connector technology can link securely to a whole range of data sources including file systems, databases, applications, APIs, flat files, and more. The PI workflow engine can carry out a whole range of data transformation tasks and can then output the data to the required destination using another secure flexible connector.

Platform Services


A Powerful Tool-set for Your Data

Data Warehouse – Securely store, update, query, and design reports on data.

File Service – Work with files, store documents, and access data within them.

Runner Service – Execute workflows on triggers or schedules in isolated threads.

Partner Service – User and role management with multiple access levels.

Logging Service – Persist logs and sync with 3rd party systems.

Workflow Manager – Manage execution and monitoring of workflows.

SFTP Server – Secure file storage for workflow inputs/outputs.

Integrations Co-Pilot - Monitor entire integration landscape.

On-Premise or Cloud - Flexible hosting to meet your needs.


Automate Complex Integrations from One Place

Creating a PI workflow couldn’t be simpler. Using a drag and drop low-code editor with a powerful set of predefined worker nodes you can create a workflow to, for example, extract data from a source system, validate it against a reusable set of rules, transform its structure or values and then send it to a destination system in a completely different file format. With PI you can work magic to clean up even the messiest data integration nightmares. Whether you choose to use PI on-premises on in the cloud, the security and integrity of your data remains a top priority.
